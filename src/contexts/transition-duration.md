## Transition Duration

### Utilities (value-driven)

| Selector    | Style                                  | Variables Required |
| ----------- | -------------------------------------- | ------------------ |
| `.duration` | `transition-duration: var(--duration)` | `--duration`       |

### Utilities (absolute values)

| Selector            | Style                          |
| ------------------- | ------------------------------ |
| `.duration-initial` | `transition-duration: initial` |
