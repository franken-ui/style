## Flex Shrink

### Utilities (value-driven)

| Selector  | Style                        | Variables Required |
| --------- | ---------------------------- | ------------------ |
| `.shrink` | `flex-shrink: var(--shrink)` | `--shrink`         |

### Utilities (absolute values)

| Selector    | Style            |
| ----------- | ---------------- |
| `.shrink-1` | `flex-shrink: 1` |
| `.shrink-0` | `flex-shrink: 0` |
