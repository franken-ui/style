## Text Indent

### Utilities (value-driven)

| Selector    | Style                                               | Variables Required |
| ----------- | --------------------------------------------------- | ------------------ |
| `.indent`   | `text-indent: calc(var(--spacing) * var(--indent))` | `--indent`         |
| `.[indent]` | `text-indent: var(--indent)`                        | `--indent`         |
