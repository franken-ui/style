## Flex Grow

### Utilities (value-driven)

| Selector | Style                    | Variables Required |
| -------- | ------------------------ | ------------------ |
| `.grow`  | `flex-grow: var(--grow)` | `--grow`           |

### Utilities (absolute values)

| Selector  | Style          |
| --------- | -------------- |
| `.grow-1` | `flex-grow: 1` |
| `.grow-0` | `flex-grow: 0` |
