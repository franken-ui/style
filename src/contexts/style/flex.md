## Flex

### Utilities (value-driven)

| Selector | Style               | Variables Required |
| -------- | ------------------- | ------------------ |
| `.flex`  | `flex: var(--flex)` | `--flex`           |

### Utilities (absolute values)

| Selector        | Style            |
| --------------- | ---------------- |
| `.flex-1`       | `flex: 1 1 0%`   |
| `.flex-auto`    | `flex: 1 1 auto` |
| `.flex-initial` | `flex: 0 1 auto` |
| `.flex-none`    | `flex: none`     |
