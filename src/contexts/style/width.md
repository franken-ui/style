## Width

### Utilities (value-driven)

| Selector | Style                                    | Variables Required |
| -------- | ---------------------------------------- | ------------------ |
| `.w`     | `width: calc(var(--spacing) * var(--w))` | `--w`              |
| `.[w]`   | `width: var(--w)`                        | `--w`              |
