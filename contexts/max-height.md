# Max Height Documentation

## Value-driven Utilities

| Selector   | Style                                             | Variables Required |
| ---------- | ------------------------------------------------- | ------------------ |
| `.max-h`   | `max-height: calc(var(--spacing) * var(--max-h))` | `--max-h`          |
| `.[max-h]` | `max-height: var(--max-h)`                        | `--max-h`          |
