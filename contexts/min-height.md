# Min Height Documentation

## Value-driven Utilities

| Selector   | Style                                             | Variables Required |
| ---------- | ------------------------------------------------- | ------------------ |
| `.min-h`   | `min-height: calc(var(--spacing) * var(--min-h))` | `--min-h`          |
| `.[min-h]` | `min-height: var(--min-h)`                        | `--min-h`          |
