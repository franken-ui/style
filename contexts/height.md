# Height Documentation

## Value-driven Utilities

| Selector | Style                                     | Variables Required |
| -------- | ----------------------------------------- | ------------------ |
| `.h`     | `height: calc(var(--spacing) * var(--h))` | `--h`              |
| `.[h]`   | `height: var(--h)`                        | `--h`              |
