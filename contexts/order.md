# Order Documentation

## Value-driven Utilities

| Selector | Style                 | Variables Required |
| -------- | --------------------- | ------------------ |
| `.order` | `order: var(--order)` | `--order`          |

## Utilities

| Selector        | Style          |
| --------------- | -------------- |
| `.-order-first` | `order: 9999`  |
| `.-order-last`  | `order: -9999` |
| `.-order-none`  | `order: 0`     |
| `.order-first`  | `order: -9999` |
| `.order-last`   | `order: 9999`  |
| `.order-none`   | `order: 0`     |
