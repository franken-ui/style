Input Time

Basic

Description

Default time input with no attributes. Should render with empty hour and minute fields in 12h format with AM/PM selector.

Expectation

Component should:

  • Display three input fields: hour (empty), minute (00), and AM/PM button (AM)
  • Hour field should accept values 1-12
  • Minute field should be disabled until hour is entered
  • Allow clearing the hour field (make it empty)
  • AM/PM button should toggle between AM and PM when clicked
  • Use colon (:) as separator between hour and minute
Status Passed
Notes -

  • Minute field should be enabled (since hour is set)

Attr: clock="24h"

Description

Time input using 24-hour clock format without AM/PM selector.

Expectation

Component should:

  • Display only hour and minute fields (no AM/PM button)
  • Hour field should accept values 0-23
  • Allow entering 00 for midnight and 23 for 11 PM
  • No AM/PM toggle functionality
Status Passed
Notes -

Attr: value="14:30"

Description

Time input with preset value in HH:MM format (24-hour). Should convert to 12-hour format display.

Expectation

Component should:

  • Display hour as "02" (converted from 14)
  • Display minute as "30"
  • Set AM/PM to "PM" (since 14:00 is 2 PM)
  • All fields should be editable
  • Form submission should return "14:30" in 24h format
Status Passed
Notes -
