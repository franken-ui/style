Basic

Description

Render without attributes. Verifies default dropdown behavior and placeholder.

Expectation

Component renders as a button that opens a calendar.

  • Button displays placeholder "Select a date".
  • Clicking the button opens the dropdown with a uk-calendar inside.
  • Selecting a date closes the dropdown (optional standard behavior) or updates the button text to the formatted date (e.g., "October 25, 2023").
